# HW2

Что нужно сделать
1. Явные Intent
   — Создайте приложение с двумя Activity.
   — На экран первой Activity добавьте поле EditText для ввода текста.
   — Также на экран первой Activity добавить кнопку "Открыть вторую Activity".
   — При нажатии передавайте строку из EditText.
   — Во второй Activity отобразите эту строку на экране.
2. Неявные Intent
   — Добавьте кнопку "Позвонить другу" на экран первого Activity.
   — При нажатии используйте неявный Intent для открытия приложения звонков (ACTION_DIAL) с номером телефона из поля EditText.
3. Системные Intent
   — Добавьте кнопку "Поделиться текстом" на экран первой Activity.
   — При нажатии отправляйте Intent (ACTION_SEND) с текстом из EditText.
   — Должно появиться системное окно «Поделиться через…».
4. Общие рекомендации к выполнению
   — Все ДЗ делается в отдельном ПРе (pull request), который НЕ будет влит в итоговый проект.
   — Не стоит забывать про валидацию данных. Если они не валидны, то как-то это обрабатывать.
